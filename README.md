just wrapped up my latest project: a fully functional Expense Tracker! 🚀

What started as a simple Python CLI tool quickly turned into an obsession with building a better user experience. I wasn't satisfied with just terminal outputs, so I pushed myself to upgrade the interface.

I built out two different GUI versions to explore different tech stacks:
1️⃣ A sleek, Dark-Mode Glassmorphism web interface using HTML, CSS, and JavaScript.
2️⃣ A native Python desktop application using the CustomTkinter library for a modern, dark-theme aesthetic.

Core Features:
🔹 Log daily expenses (Date, Category, Description, Amount)
🔹 Dynamic dashboard for viewing transaction history
🔹 Automatic total spending calculation
🔹 Clean, error-proof input validation

Building this really solidified my understanding of control flow, data structures, and connecting backend logic to front-end interfaces.
